examples:

A static website is just that - static. It is not dynamic. It does not change (frequently).

An example might be a Restaurant's menu, but without order processing.

text - anything written in plain English, or whatever language.

images/logos/videos

just about anything seen on the internet can be expressed as a single, static web page.
